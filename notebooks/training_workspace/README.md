# Training Workspace

Use `SQLI_Model_Training_Clean.ipynb` for the model-training workflow.

This notebook keeps only the pipeline used in the project documentation:

- dataset loading and cleaning
- SQL-aware preprocessing
- TF-IDF character n-gram features
- Random Forest, XGBoost, and Hybrid Ensemble training
- evaluation metrics and confusion matrices
- model artifact export for the backend

Extra narrative notes, dashboard-only logic, and unused experiment sections are intentionally excluded from this clean notebook.
